This is aimed at producing a live CD and/or Virtual Machine installer:
- based on Debian Linux
- with the FreePascal (FPC) compiler and
- Lazarus IDE

Nice to have:
- update FPC, Lazarus using e.g. fpcup or - more supported - stock SVN update/make
