Code for importing bank transaction data in various formats

Input format:
- ABNAmro tab-separated text
- ING comma-separated text
- Sberbank text

Additions for more import formats are welcome

banco is a GUI for importing/exporting. If you want to use Firebird or SQLite export format, please make sure the relevant libraries are installed (Linux) or present in e.g. the application directory (Windows)

bankconvert is a text-based/batch oriented conversion tool. It uses a demo "screen" output format. Additional output formats can be easily created
